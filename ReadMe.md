# University Projects Repository

This repository contains six game/VR projects developed during university coursework.

## Repository Summary
- Projects 1 to 3 are practice Unity projects used to learn core 2D game development concepts.
- Project 4 and VR Art Gallary are final semester projects with larger scope.
- VR web small projects is a collection of browser-based VR practice pages.

## Projects
1. project1-number guess
	- Practice project: number guessing game with scene flow and basic UI logic.
	- README: project1-number guess/README.md
2. project2-flappy bird
	- Practice project: flappy-style arcade game with obstacles, score, health, and win/lose flow.
	- README: project2-flappy bird/README.md
3. project3-polic car game
	- Practice project: lane-based shooting game with enemy spawning, score, health, and pause menu.
	- README: project3-polic car game/README.md
4. project4-Lab_project-Delivery Car
	- Final semester project: delivery-driving game with order system, money progression, and objective guidance.
	- README: project4-Lab_project-Delivery Car/README.md
5. VR web small projects
	- Practice collection: small A-Frame pages for VR/web learning and experimentation.
	- README: VR web small projects/README.md
6. VR Art Gallary
	- Final semester project: web-based VR art gallery and portfolio experience built with A-Frame.
	- README: VR Art Gallary/README.md


## Notes
- Unity projects can be opened directly in Unity Hub.
- Browser VR projects can be run by opening their HTML files in a modern web browser.
