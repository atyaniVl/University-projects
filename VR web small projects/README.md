# VR Web Small Projects

A collection of small A-Frame practice pages created while learning browser-based VR development.

## Project Type
Practice project collection

## Overview
This folder contains multiple independent HTML experiments that explore A-Frame fundamentals such as:
- Entity creation and placement
- Materials and textures
- Lighting and sky/ground setup
- Basic animations and hover interactions
- Importing 3D models (GLTF/OBJ)

## Key Demo Files
- introToAframe.html
  - Intro scene with basic A-Frame entities and layout.
- chapter5.html, chapter 6.html, Chapter6_revision.html, chapter7.html
  - Iterative lessons and experiments for interaction, animation, text, and model loading.
- test.html
  - Additional model-loading and scene validation page.

## Technologies
- HTML/CSS/JavaScript
- A-Frame (multiple version links used across files)
- Optional A-Frame environment component in some demos

## Assets
This folder includes many local assets for experiments:
- 3D models (OBJ, GLTF, and related material files)
- Textures and images
- Audio file(s)

## How To Run
1. Open any .html file directly in a browser.
2. Start with introToAframe.html or chapter7.html.
3. Navigate between files to view different mini-project examples.

## Learning Focus
- Building VR scenes directly on the web
- Fast prototyping of interactions
- Practicing asset import and scene composition
