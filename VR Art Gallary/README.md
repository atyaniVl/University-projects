# VR Art Gallary

Final semester web-VR project built with A-Frame.

## Project Type
Final semester project

## Overview
VR Art Gallary is an interactive virtual portfolio/gallery experience presented in the browser. It includes:
- An opening landing page with social links and a launch button
- A main A-Frame VR scene showing my artwork and 3D models
- Interactive hover/click behaviors for object descriptions and floor transitions
- Rich texture usage for floors, walls, and props

## Main Files
- openning page.html
  - Landing page with background image, Instagram/YouTube links, and a button to open the VR scene.
- artGallerySystem.html
  - Main A-Frame gallery system with assets, environment, camera/cursor controls, textured architecture, and interactive exhibits.

## Technologies
- HTML/CSS/JavaScript
- A-Frame (aframe.min.js)
- A-Frame environment component
- A-Frame event-set component

## Interaction
- Use mouse cursor for look-and-click interactions.
- Click interactive doors/objects to trigger scene movement and reveal descriptions.
- Explore exhibited models and images in the virtual gallery space.

## Assets
The project uses multiple local assets, including:
- 3D model folders under 3d objects/
- Texture libraries for floors/walls/doors
- UI/social images and background visuals
